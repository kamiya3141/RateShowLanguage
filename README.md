# RateShowLanguage
冷笑系プログラミング言語 RateShow です。BF系です。

<h1>・言語仕様</h1>


<h3>・"きちーww" -> ポインタの値を1増やす</h3>

<h3>・"ええてw" -> ポインタの値を1減らす</h3>

<h3>・"うおw" -> ポインタを右にずらす</h3>

<h3>・"どわー笑" -> ポインタを左にずらす</h3>

<h3>・"ちょwお前必死やんww" -> ポインタの値を出力する</h3>
